# Ecommerce-Data-Modeling-PowerBI
Designed and implemented a relational data model using Customers, Orders, Products, Sellers, Reviews, Payment, and Delivery datasets. Established one-to-many relationships and applied star schema concepts for efficient reporting. Performed data transformation and cleaning using Power Query. 


## Project Overview

This project demonstrates data modeling techniques in Power BI using an E-commerce dataset. Multiple tables were integrated to create a relational data model for reporting and analysis.

## Dataset Tables

- Customers
- Orders
- Products
- Sellers
- Reviews
- Payment Mode
- Delivery Status

## Data Modeling Concepts

- Fact and Dimension Tables
- One-to-Many Relationships
- Star Schema Design
- Data Transformation using Power Query
- DAX Measures

## Skills Demonstrated

- Data Modeling
- Power BI
- Power Query
- DAX

- Relationship Management

## Tools Used

- Power BI Desktop

## Project Outcome

Created an optimized data model to support business reporting and enable efficient analysis of customer behavior, orders, products, payments, and delivery performance.
